SUPPLIES

These are the most necessary supplies you will need if you follow an architecture uni:

1. Basic Drawing Tools (First Year Essential)

  • Sketchbook (A3 often required) Mechanical pencils + graphite pencils (HB–4B) Eraser + sharpener Ruler (30–50 cm) Set square / triangle Compass Technical pens / fineliners These are used for hand drawing, technical drafting, and concepts.

2. Model-Making Materials

  • Cutting mat Precision cutter (X-Acto type) Metal ruler Glue (UHU / wood glue) Cardboard, foam board, balsa wood Architecture programs require physical models, especially in early years.

3. Laptop & Software

  • Very important after first year. Strong laptop (good RAM + graphics card) Software like AutoCAD, Revit, SketchUp, Rhino, Adobe Suite Rendering and 3D modeling become essential.

4. Studio Essentials

  • Large portfolio case (for presentations) USB drive / external hard drive Notebook for concepts Desk lamp (if working in dorm/apartment)
